# Official Ray VS Code Extension (`@ray/vscode-extension`)

The `@ray/vscode-extension` package provides first-class developer tooling for Ray projects directly inside VS Code.

## Features

- **IntelliSense**: Auto-complete configuration keys in `ray.config.ts`, `ray.config.js`, and `ray.config.mjs`.
- **Real-time Diagnostics**: Detect unknown properties, type mismatches, or syntax issues as you type.
- **Hover Documentation**: View descriptions, types, and defaults directly in the editor.
- **Command Palette shortcuts**:
  - `Ray: Start Dev Server`
  - `Ray: Build Project`
  - `Ray: Run Benchmark`
  - `Ray: Migrate Project`
  - `Ray: Create Project`
  - `Ray: Open Benchmark Report`
